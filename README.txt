1. Create Database in localhost/phpmyadmin with Database name of hots_stats

2. Load http://localhost/hots_stats/CreateDB.php
http://localhost/hots_stats/createDB.php

3. To populate DB use back_end.php, can refresh ~2 times per minute before too many requests gets called
http://localhost/hots_stats/back_end.php

4. To see stats on webpage load index.html
http://localhost/hots_stats/index.html
